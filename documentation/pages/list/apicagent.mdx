## Get User-Agent Information (GET)

Fetches information about the provided user-agent string via a GET request.

- **URL**: `/user-agent`
- **Method**: GET
- **Headers**:
  - `User-Agent` (required): The User-Agent string sent by the client.
- **Response**: Returns a JSON object containing information about the user-agent.

**Example Request:**

```plaintext
GET /user-agent
